# UXDG 320 — React Exercises

Six exercises that teach React mechanics by building a designer portfolio app.

## Running an exercise

```bash
cd exercises/01-jsx-components/starter
npm install
npm run dev
```

## Exercises

| # | Concept | Folder |
|---|---------|--------|
| 1 | JSX & Function Components | `exercises/01-jsx-components/` |
| 2 | Props | `exercises/02-props/` |
| 3 | Events | `exercises/03-events/` |
| 4 | State (useState) | `exercises/04-state/` |
| 5 | Lists & Keys | `exercises/05-lists-keys/` |
| 6 | useEffect + Lifting State | `exercises/06-useeffect-lifting/` |
